# Tech Stack

## Runtime & Framework
- PHP 8.3 (container), requires 8.2+
- Symfony 7.1
- Twig templating
- Symfony Forms + Validator
- Symfony Mailer

## PHP Extensions Required
- `ext-gnupg` — GnuPG bindings for server-side signing
- `ext-openssl` — token encryption (AES-256-CBC)
- `ext-ctype`, `ext-iconv`, `ext-posix`, `ext-zend-opcache`

## Frontend
- Stimulus (`@symfony/stimulus-bundle`)
- Turbo (`@symfony/ux-turbo`)
- Symfony AssetMapper (no Webpack/Vite)
- Bootstrap (vendored in `public/bootstrap/`)
- OpenPGP.js — client-side encryption (loaded via `importmap.php`)

## Infrastructure
- Docker + Docker Compose (3 services: `php`, `nginx`, `mailhog`)
- NGINX reverse proxy — config in `docker/nginx/`
- PHP-FPM — config in `docker/php/`; runs as `www-data`
- MailHog — local mail catcher (SMTP: 1025, UI: http://localhost:8025)

## Testing
- PHPUnit 9.5
- Symfony PHPUnit Bridge
- Config: `phpunit.xml.dist`, test env vars: `.env.test`

## Environment Variables
- `APP_SECRET` — derives the token encryption key; must be set to a random string
- `MAILER_DSN` — SMTP DSN; default `smtp://mailhog:1025` works with the Docker setup
- `PGP_PRIVATE_KEY_PASSPHRASE` — passphrase for the server PGP private key
- `GNUPGHOME` — set to `/var/www/app/config/pgp` in `docker-compose.yml`

Copy `.env.example` to `.env` before first run. Never commit `.env`.

## First-Time Setup

```bash
cp .env.example .env
docker-compose up -d
docker exec php composer install

# Generate server PGP key pair (creates config/pgp/private.key + public.key)
docker exec php bash /var/www/app/scripts/init-pgp.sh

# Fix ownership so www-data (PHP-FPM) can read keys and write cache
docker exec php bash -c "chown -R www-data:www-data /var/www/app/var/ /var/www/app/config/pgp/"
```

## Common Commands

```bash
# Start containers
docker-compose up -d

# Install PHP dependencies
docker exec php composer install

# Clear Symfony cache
docker exec php php bin/console cache:clear

# Reinstall JS importmap assets
docker exec php php bin/console importmap:install

# Run tests
docker exec php php bin/phpunit --no-coverage

# Run a specific test file
docker exec php php bin/phpunit tests/BootstrapTest.php --no-coverage

# Tail logs
docker exec php tail -f var/log/dev.log
```

## PGP Key Notes
- Keys are generated by `scripts/init-pgp.sh` (runs inside the container)
- `scripts/docker-setup-pgp.sh` is designed to be run from the host — it shells into the container internally
- Default key generation uses `%no-protection` (no passphrase); `PGP_PRIVATE_KEY_PASSPHRASE` can be left as placeholder
- Key files live in `config/pgp/` which is gitignored
- Required permissions: `private.key` → 600, `public.key` → 644, `config/pgp/` → 700, owner `www-data`
